Existen dos formas de establecer el precio de venta y apartado. Por default se calcula en base a la configuracion establecida en la opción "Porcentaje sobre avaluo para venta y apartado".

En esta opción se permite establecer el precio de venta, ya que en ocaciones esto es requerido. Por ejemplo cuando un cliente regatea el precio de un producto, un producto esta dañado y ya no puede ser vendido con los parametros establecidos en la opcion "Porcentaje sobre avaluo para venta y apartado" o cualquier otro escenario que se presente.

Una vez que se actualize el precio en esta pantalla, ya no sera considerada la configuración de la pantalla "Porcentaje sobre avaluo para venta y apartado".
